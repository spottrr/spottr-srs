# Spottrr Software Requirements Specification

SRS document for the Spottrr senior project, written in LaTeX.

## Prerequisites

- TeX distribution with LaTeX2e support (e.g., [MiKTeX](https://miktex.org/))
- [VS Code](https://code.visualstudio.com/) with [LaTeX Workshop](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop) extension (recommended)

## Getting Started

1. Open `main.code-workspace` in VS Code
2. Edit project variables in `document.tex`
3. Write content in the `body/` directory
4. Build to generate `document.pdf`
